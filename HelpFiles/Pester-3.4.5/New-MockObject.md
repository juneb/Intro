# New-MockObject

## SYNOPSIS
This function instantiates a .NET object from a type. The assembly for the particular type must be
loaded.

## SYNTAX

### Set 1
```
New-MockObject [-Type] <Type> [<CommonParameters>]
```

## EXAMPLES

